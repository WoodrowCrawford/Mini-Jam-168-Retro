## Story
You play as a boy who only has one goal: to watch TV! However, things arent what they seem in this short visual novel story!


## Technical Implementation
- **Game Boy ROM** — Built using GB Studio, compiled to a native .gb ROM file compatible with Game Boy emulators and real hardware
- **Hardware Deployment** — Successfully deployed and tested on a 3DS via Game Boy ROM loading
- **Hardware Constraints** — Designed within strict Game Boy limitations including 160x144 resolution, 4 color palette, and limited memory
- **Web Deployment** — Deployed as a browser playable version using GB Studio's web export and hosted via GitHub Pages


## Tech Stack
- GB Studio
- Game Boy ROM (.gb format)
- GitHub Pages (web deployment)
